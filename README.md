# 📱 Hotspot SSH Scanner (Native Android)

A professional, high-performance Android utility designed to discover active devices on a local network (Hotspot or WiFi) and connect via SSH with a single tap.

---

### 🚀 Key Features
* **Active Network Discovery:** Scans your local network to identify devices with **Port 22 (SSH)** open.
* **One-Click Connection:** Automatically generates a connection button for found devices.
* **Native Performance:** Built with **Kotlin/Java** for maximum speed and minimal battery drain.
* **Custom Build System:** Compiled and signed using a custom terminal-based workflow (no IDE bloat).

---

### 🛠 Technical Showcase (The "Hardcore" Way)
Unlike most modern apps, this project was developed using a **Text Editor & Terminal** approach to ensure a clean, optimized codebase:
* **Manual Gradle Management:** Custom `build.gradle` and `settings.gradle` configurations.
* **Automated Workflow:** Uses `build_and_sign.sh` for automated compilation, alignment, and APK signing.
* **Low-Level Networking:** Direct Socket implementation for fast port scanning.

---

### 📁 Project Structure
* `app/src/main/java/`: Core Kotlin logic for scanning and SSH handling.
* `app/src/main/res/`: Native XML layouts and Material Design assets.
* `build_and_sign.sh`: My custom script to build the APK from the Linux Mint terminal.
